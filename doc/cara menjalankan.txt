1. Install netbeans.
2. Tambahkan library mysql-connector.jar ke dalam folder project
3. Install MySQL.
4. Nyalakan MySQL.
5. import database ke MySQL.
6. buka file src\java\springapp\db\DBConnector.java. Atur konfigurasi username, password dan nama database.
7. Buka netbeans.
8. Run program.
9. Enjoy
